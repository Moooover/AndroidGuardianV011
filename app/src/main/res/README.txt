Unzip this package directly inside your Android res folder:

app/src/main/res/

Included:
- mipmap-mdpi/ic_launcher.png
- mipmap-hdpi/ic_launcher.png
- mipmap-xhdpi/ic_launcher.png
- mipmap-xxhdpi/ic_launcher.png
- mipmap-xxxhdpi/ic_launcher.png
- ic_launcher_round.png for each density
- ic_launcher_foreground.png for each density
- mipmap-anydpi-v26 adaptive icon XML configs

PNG images are generated from the uploaded icon.
